응급처치
1. 물집이 심하게 아프지 않다면, 터지지 않도록 유지하세요.
2. 물집을 덮고 있는 손상되지 않은 피부는 세균으로부터 자연스러운 방어막이 되어 감염 위험을 줄여줍니다.
3. 물집은 붕대나 밴드로 덮어 보호하세요.
4. 물집 통증을 줄이려면, 물집 위의 피부를 제거하지 않은 채로 내부 액체만 배출하세요.

물집 배출 방법
1. 손과 물집 부위를 비누와 물로 깨끗이 씻습니다.
2. 물집에 소독제를 바릅니다.
3. 소독용 물티슈 또는 알코올로 바늘을 소독합니다.
4. 물집 가장자리에 바늘로 몇 군데 작은 구멍을 내어 액체를 배출합니다.
   이때, 물집의 윗피부는 제거하지 마세요.
5. 항생제 연고 또는 바셀린을 바르고, 비접착성 붕대나 거즈 패드로 덮습니다.
6. 며칠 후, 죽은 피부는 핀셋과 가위를 소독한 후 조심스럽게 제거하세요.
7. 연고를 다시 바르고 붕대를 교체한 후, 매일 감염 여부를 확인합니다.

주의 사항
당뇨병, 혈액 순환 장애, 감염 이력이 있는 경우 감염 예방에 특히 주의해야 합니다.
